Welcome to the AI Chat Widget Demo!

This page demonstrates the embedded AI chat widget functionality. The chat widget should appear in the bottom-right corner of your screen.

Note: Make sure your backend server is running on http://localhost:8000 and frontend on http://localhost:3000 for the widget to work properly.

üìã How to Embed

To embed this chat widget on your website, simply add the following script tag to your HTML:

<script src="http://localhost:8000/widget-loader/test-widget-key-1.v1.js"></script>

The widget will automatically:

  • Load and position itself in the bottom-right corner
  • Fetch configuration from your backend
  • Apply custom styling and branding
  • Handle chat conversations with rate limiting

The widget can be configured through the admin dashboard at http://localhost:3000/widgets.

Available configuration options:

  • Accent Color: Primary color theme
  • Logo URL: Company logo (optional)
  • Active Status: Enable/disable the widget
